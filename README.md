# barry-harris-exercises

[Barry Harris](https://en.wikipedia.org/wiki/Barry_Harris) had a special style of jazz piano that was quite special. [Lilypond](https://lilypond.org/) makes it easy to generate nice music charts.

The code in this repo creates Barry Harris style exercises for jazz piano practice. 
See [BH-scales.pdf](BH-scales.pdf)

Harmonies and voicings are easy to goof up. Please create an [issue](https://github.com/aaelony/barry-harris-exercises/issues) if you discover any mistakes.
